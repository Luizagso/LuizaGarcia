🚀 **Acesse o portfólio online:** [https://luizagso.github.io/LuizaGarcia/tailwind/index.html](https://luizagso.github.io/LuizaGarcia/tailwind/index.html)

Este projeto consiste na construção do meu portfólio pessoal e profissional, focado em demonstrar minha vivência nas verticais de Gestão de Projetos (Product Owner) e Desenvolvimento Web Fullstack. O repositório possui fins extras de portfólio acadêmico, aplicando conceitualmente padrões modernos de mercado em estruturação `Web (HTML)` e interatividade de layout responsivo.

## 🛠️ O Diferencial do Repositório (Duas Abordagens)

Com o objetivo prático de comparar implementações e demonstrar domínio real no front-end, este projeto foi programado contendo a **mesma página de portfólio** idealizada sob duas versões isoladas em diferentes bibliotecas CSS de mercado:

*   **Versão Bootstrap 5 (`/bootstrap/`):** Utiliza as classes clássicas do framework baseadas em estrutura. Total adesão ao ecossistema da biblioteca, fazendo uso do JavaScript compilado do próprio _Bootstrap_ para a renderização limpa do **Dark Mode** e acionamento interativo dos Modais.
*   **Versão Tailwind CSS (`/tailwind/`):** A mesma ideia reconstruída totalmente usando o conceito *Utility-First*. Design mais customizado, *hover* interativo nas malhas, e um **JavaScript Vanilla (Puro e Artesanal)** que regeu completamente as nuances modais da página, além de capturar com persitência a lógica base de background em CSS Puro de classe `[dark]`.
